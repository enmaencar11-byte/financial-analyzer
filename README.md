\# FinAnalyzer Pro



\*\*ES\*\* | Sistema automatizado de análisis financiero aplicado a emisores

del mercado de valores dominicano (SIMV).



\*\*EN\*\* | Automated financial analysis system applied to issuers listed

on the Dominican securities market (SIMV).



\---



\## ¿Qué hace? | What does it do?



\- 30+ financial ratios under IFRS (liquidity, profitability, leverage, activity)

\- 3-level DuPont analysis

\- Altman Z''-Score adapted for the Dominican market

\- CAPM and WACC with country risk premium (Damodaran methodology)

\- EVA (Economic Value Added)

\- Interactive dashboard (Streamlit)

\- Automated 8-page PDF report



\---
